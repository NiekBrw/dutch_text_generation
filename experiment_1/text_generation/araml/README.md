## ARAML

#### Dependencies
```sh
  tensorflow-gpu == 1.15
  numpy
  scipy
```

#### Summary
Train the ARAML method on Dutch tweet data.

Task for testing: generate 4000 samples.

#### How to run
```sh
  python araml_lm.py --task_name covid
```
